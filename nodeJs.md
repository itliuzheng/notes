## 启动服务

```bash
##全局安装nodemon命令
npm install nodemon-g 
## 启动应用
nodemon /app.js
## 重启
自动重启
## 停止
ctrl + C
```



# Node.js项目

## 前端(Frontend)

- 前端工程化环境（webpack）

- CSS预处理工具（scss）

- JS库：JQuery

- SPA：single page application，路由：SME-Router

- UI组件库：Bootstrap（AdminLTE）

## 后端(Backend)

- Node.js
- Express
- MongoDB(Mongoose)
